A beautiful, fully functional calculator with a pink theme built using HTML, CSS, and JavaScript. This project demonstrates front-end web development skills with a focus on user interface design, DOM manipulation, and mathematical operations.

.Description

This is a stylish pink-themed calculator web application that performs all basic arithmetic operations. The calculator features a modern, visually appealing design with a pink gradient background and smooth user interactions. It handles complex calculations with proper operator precedence and includes error handling for edge cases like division by zero.

. Features

- Basic Arithmetic Operations**: Addition (+), Subtraction (-), Multiplication (*), Division (/)
- Beautiful Pink Theme**: Eye-catching gradient design with multiple shades of pink
- Decimal Support**: Perform calculations with decimal numbers
- Chain Calculations**: Continue calculating without pressing equals after each operation
- Error Handling**: Displays "Error" message for invalid operations (like division by zero)
- Clear Function**: Reset calculator with a single click
- Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- Smooth Animations**: Active button states for better user feedback
- Smart Display**: Automatically resets after calculation results

2. Click an operator (+, -, *, /)
3. Click more numbers

5. Press "C" to clear and start over
